# SnakeGame
THIS IS SNAKE GAME USING JavaScript

1 USE LEFT RIGHT UP DOWN KEYS TO CONTROL THE SNAKE

2 IF YOU HIT WALL OR YOURSELF YOU DIE

3 PRESS ANY KEY TO RESTART

4 DO NOT EDIT THIS BRANCH!!!
